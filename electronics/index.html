Electronics

RSP-1 · COMPLETED

Building a Nintendo emulator powered by Raspberry Pi was something we considered to be a fun project more than anything else. Being relatively cheap, the brains of the device is a Raspberry Pi model B, encased in a 3D printed enclosure. While this project was mostly for fun, it serves as valuable experience with the Pi, which we plan to incorporate in future projects, as well as experience with soldering and wiring.
